# task-2
 My Portfolio (HTML Project)
This is a small portfolio webpage I made using only HTML.
It’s a practice project to show basic info about me, my studies, and my art.

 What’s Inside
Title banner with a moving marquee saying PORTFOLIO

A table layout with my details (name, college, year, Insta ID, etc.)

My photo displayed inside the page

Simple use of colors (black background, white/yellow/orange text)

 Tech Used
HTML (basic tags like <table>, <font>, <marquee>, <img>)

No CSS file, all styling is inline

No JavaScript

 Files
portfolio.html → main webpage file

TUSHAR.jpeg → my photo used in the page

 How to Run
Keep both files (portfolio.html and TUSHAR.jpeg) in the same folder.

Open portfolio.html in any browser.

You’ll see my portfolio page with details and image.
 Purpose
This project is just for practice and learning.
It’s not a professional portfolio, but a simple HTML design to understand tags and layouts.
